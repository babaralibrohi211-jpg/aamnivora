# Aamnivora Website

Premium Next.js App Router website for Aamnivora, a digital transformation company focused on local businesses.

## Stack

- Next.js 16 App Router
- React 19
- TypeScript
- Tailwind CSS
- Framer Motion
- React Hook Form
- Zod
- Lucide React

## Commands

```bash
npm install
npm run dev
npm run build
```

## Notes

- Images load from Unsplash and are configured in `next.config.mjs`.
- Contact form opens a `mailto:` flow and works without a backend.
- Navigation is mobile-first and animated.
